# G1 / C3 post-interval-bulk consumer reassessment

Date: 2026-07-16

Status: **local numerical prerequisites are present; bounded optical-lineage
composition design reassessment selected; no composer source authorized.**

## Decision

The verified conditional cell step, interval-incident interface event and
one-band conditional bulk transfer now supply the three local numerical
operations that were missing from the earlier refracted-path composition
readiness correction. This closes the reason to revisit the composer design;
it does not make the old composer proposal implementation-ready.

The smallest dependency-correct next package is a **bounded optical-lineage
composition mathematical design reassessment**. It must inventory the exact
current public seams, specify immutable ordered bindings between their receipts
and decide whether a minimal additive composer can replay one transmitted lane
without copying or broadening any local numerical kernel.

The reassessment is read-only. It cannot add a crate, schema, dependency,
adapter or production algorithm, and it cannot treat the earlier candidate
budgets as approved limits.

## Exact seam questions

The next reassessment must answer these before any readiness claim:

1. What canonical lane identity binds profile, reconstruction, spectral band,
   initial state, every cell-step event, every bulk transfer and every
   interface event in exact order?
2. How is the certified hit-point box copied into the next cell-step input
   without caller narrowing, and which source/revision identity proves that
   derivation?
3. How is one `BandInterfaceEventV1.transmitted_direction_xyz` converted
   losslessly between the interface owner's Q1.62 decimal codec and the
   physical owner's Q1.62 decimal codec?
4. Which exact face, source cell, target cell and face-interaction identities
   must match before an interval interface event may follow a cell-step event?
5. How are per-cell transmission intervals combined with per-interface
   transmitted-power intervals using outward arithmetic without duplicating
   the bulk or interface kernels?
6. Which typed terminations apply to outer exit, unavailable current or
   neighbour, all-TIR, ambiguous interface branch, ambiguous next face,
   no-forward-progress, nonconvergence, repeated state and work exhaustion?
7. What endpoint relation can be stated without inventing an exact receiver
   point from a point box? If no sound bounded endpoint certificate exists,
   the composer must remain opportunity-only.
8. Which event, byte, memory and arithmetic ceilings survive hostile
   three-band dispersion and repeated-interface portfolios on PC and mobile?

## Candidate comparison required

The design reassessment must compare at least:

- a thin immutable lineage manifest containing hashes/IDs of separately
  replayable local receipts;
- nested complete local objects inside one capped result;
- a streaming fold with a replayable canonical transcript; and
- no composer, retaining only manual local evidence.

The leading candidate is the thin lineage manifest because the local objects
already have strict codecs and identities. It is not selected until the audit
proves that replay has no ambient lookup, ordering ambiguity or unbounded
history and that deletion leaves every local module unchanged.

## Permanent boundaries

- One post-interface band is one lane; RGB geometry cannot be recombined.
- The composer may orchestrate and bind but may not implement traversal,
  Snell/Fresnel arithmetic, extinction, exponential transfer or endpoint
  classification on behalf of an owner.
- A typed local ambiguity cannot be repaired by midpoint, epsilon, corner,
  normalization or platform-native float.
- Attenuation through a certified current cell precedes outer or unavailable
  terminal disposition.
- Interface private arithmetic is not migrated merely to simplify composition.
- Receiver physiology, detectability, perception and rendering remain
  downstream; optical opportunity is not visibility.
- Categorical occupancy or material identity cannot paint biome presentation
  seams; continuous causal fields and sharp-cause evidence remain authoritative.

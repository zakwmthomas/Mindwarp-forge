# Grounded Search Case Result

**Classification:** deterministic capability-free disposable simulation.
**Effect:** evidence only; no production, worker-selection, authority, runtime,
network, filesystem, process, clock, randomness or game-state capability.

## Contract tested

The harness models a domain-owned finite graph of probes. Each probe has a hard
validity result, objective measurement, fallible target-resistance estimate and
typed neighbours. It compares exhaustive enumeration, local greedy descent and
a bounded grounded beam. A grounded result freezes a trace and candidate ID;
it does not bake an asset, execute work or promote anything.

## Case portfolio

| Case | Compared methods | Result | Scope decision |
|---|---|---|---|
| Semantic deceptive minimum | Greedy, exhaustive, grounded beam | Greedy stops at a locally attractive dead end; bounded beam follows a worse intermediate state and reaches the accepted candidate | Domain-local pilot candidate for proposal ordering only |
| Forge Windows diagnosis | Exhaustive, grounded beam | Both find the target; grounded ordering uses fewer probes in the synthetic retained-failure fixture | Cross-domain method candidate; requires real diagnosis-local trials |
| Organism morphology | Grounded beam | Reaches the synthetic acceptance envelope | Case-local only; no anatomy, ecology or visual-quality claim |
| Spatial network | Grounded beam | Reaches the synthetic acceptance envelope with an independent adapter | Domain-local candidate; compare against graph algorithms later |
| Misleading resistance | Exhaustive, grounded beam | The resistance estimate sends the bounded method away from the only accepted candidate; exhaustive search succeeds with sufficient budget | Rejected transfer; proves the heuristic is not universal |
| Hard constraint | Grounded beam | An objectively attractive invalid node cannot ground; the valid candidate is retained | Required fail-closed behavior |
| Budget/malformed input | Grounded beam | Returns indeterminate or invalid rather than success | Required fail-closed behavior |
| Replay | Grounded beam | Byte-level trace fingerprint is deterministic and method-specific | Protocol evidence only |
| Selective aging state | Pure presentation function | Juveniles remain juvenile; eligible adults/elders display adult while locked; disabling reveals continuing age | Confirms the design formula, not a full aging system |

Eight Rust tests pass. The cases deliberately use tiny integers and synthetic
graphs so logical defects are found before any expensive PC, renderer, AI,
physics or engine run.

## What the evidence supports

- A small shared trace vocabulary can compare bounded search methods without
  centralizing domain meaning.
- Goal-grounded search can beat greedy descent in a deceptive case and can
  reduce probes when a target-local resistance estimate is informative.
- Hard validity, budgets, deterministic replay and negative-transfer recording
  are non-negotiable.
- Successful morphology and routing fixtures demonstrate adapter shape, not
  product fitness.
- The common **experimental protocol** is a universal-protocol candidate.

## What the evidence rejects or leaves open

- The grounded-beam heuristic is not universal; the misleading-resistance case
  defeats it.
- Natural provenance does not make a method correct, efficient or applicable.
- No general AI planner, semantic generator, organism generator, pathfinder,
  field solver, SDF engine, P2P network, commerce layer or runtime behavior has
  been implemented.
- No case proves visual quality, anatomy, physical validity, mobile performance,
  high-end fidelity, infinite scale or bandwidth savings.
- Search parameters and target-resistance functions remain domain-owned and
  require target-local baselines under the Federated Improvement Kernel.

## Gate decision

Approve the method-library policy and retain the harness as a proof tool. Do
not connect it to semantic construction, Forge research, worker selection or
game generation yet. The next active product work remains the preserved H5
visual intake after this evidence is presented and the full Forge gate passes.

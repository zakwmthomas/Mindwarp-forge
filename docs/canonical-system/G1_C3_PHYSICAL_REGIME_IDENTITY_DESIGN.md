# G1 C3 physical-regime identity design

Status: **bounded implementation-ready design**.

The exact environmental-opportunity graph already carries the continuous
regional exposure/moisture combination and its physical media. Adding named
regime bands would duplicate that evidence and introduce arbitrary thresholds.

The missing consumer seam is identity separation. The graph fingerprint is
place/provenance sensitive because node identities bind the world packet. A
second identity will hash only the canonical sorted physical opportunity
values. Physically identical regions may therefore share a regime identity
while retaining distinct packet, graph and lineage identities.

The physical-regime identity is not a biome, similarity metric, quality score
or evidence of interchangeability. Exact equality only is supported. Changing
exposure, moisture, media or signals changes it; changing provenance alone does
not. Macro-lineage candidates retain both the exact graph reference and this
physical-regime reference.

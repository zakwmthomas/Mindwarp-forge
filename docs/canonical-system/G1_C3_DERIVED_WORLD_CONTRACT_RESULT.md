# G1 C3 derived-world contract result

Status: **prototype tested; C3 remains incomplete against the full master-plan scope**.

## 2026-07-15 causal-integrity correction

An adversarial review found that downstream crates accepted a public packet by
schema and packet-ID shape without replaying its exact source input. Packet
content could therefore be fabricated in memory and still drive addressable
world and sensory-support outputs. The repaired contract canonicalizes signal
source order, validates packet content before serialization, and requires every
downstream causal consumer to receive the exact `WorldGenerationInput` and pass
`validate_world_packet`. Regression tests reject foreign-input packets,
fabricated packet content, fabricated opportunity nodes, and input-order
identity drift.

## Implemented boundary

The first real `derived-world-rules` module now supplies a strict,
capability-free causal seam from exact reconstruction, physical-chain and
regional field evidence
to later hierarchy, ecology, sensory and aesthetic work. It deliberately
compiles only:

- physical three-band palette bounds caused by stellar irradiance,
  atmospheric transmission and dominant material reflectance;
- environmental signal availability from bounded baseline potentials,
  regional visible exposure and required atmosphere/liquid/substrate media,
  with no propagation or distance-attenuation claim; and
- canonical input/output identifiers, limitations and authority-negative
  evidence.

It contains no organism, biome, beauty score, harmony rule, shader, asset,
runtime object or natural-solver selection. A valid packet can have no visible
light while retaining magnetic or chemical signals, so later biology is not
forced to assume eyes or Earth senses.

## Cheap and integrated proof

Eight capability-free cases pass:

1. deterministic canonical compile, decode and replay;
2. palette changes only through physical star/atmosphere/material causes;
3. pressure, chemical and vibration signals fail without their required media;
4. nonvisual signal ecology remains valid when visible radiance is absent;
5. invalid ranges, duplicate channels, unknown solver fields and noncanonical
   bytes fail closed;
6. reconstruction identity changes provenance without fabricating a physical
   difference when physical drivers are unchanged;
7. permuted signal-source input compiles to the same canonical packet; and
8. fabricated public packet content fails before serialization.

A Forge Desktop integration fixture stores the exact canonical input and output
through the existing read-only ProofReceipt projection and Reference Studio.
Kernel object, event and candidate counts remain unchanged by recording and
inspection. Repository-wide module counts and the complete Forge gate are
re-evaluated separately and are not frozen in this result document.

## Retained limits and next proof

This is a synthetic causal reference, not scientific planet validation. Field
quality/cost expansion, second-platform evidence, broader physical ranges,
regional variation, biome/niche inputs and any diffusion/SDF/Voronoi/branching
candidate remain open. C3 therefore stays active for a bounded quality, causal

## Failures found and repaired

1. Generated worker-feedback hashes depended on checkout line endings.
   Canonical UTF-8 text hashing now normalizes CRLF/CR to LF and has an
   LF/CRLF-equivalence regression fixture.
2. The modularity fixture matched formatted error text too rigidly and failed
   when a long worktree path wrapped the message. It now permits intervening
   whitespace while still requiring both module violations.
3. Byte-hashed immutable JSON evidence was vulnerable to Git text conversion.
   `evidence/** -text` now preserves exact bytes, the policy is role-classified,
   and the historical loader receipt blob is retained with its verified hash
   `9e6d266b518438760e11d6c6158bab6c930dde1fbc6e97ad5aa467365d73968e`.
4. The offline PE parser self-test depended on ignored prebuilt target output.
   It now builds its inert parser fixture from committed source with locked,
   offline Cargo when absent; the proof parser itself still cannot invoke Cargo
   or execute the candidate.

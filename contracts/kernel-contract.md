# Forge Kernel Contract v0.1

The Kernel is the protected trust boundary. It owns immutable evidence, content-addressed objects, append-only events, candidate state, explicit authority checks, and deterministic replay inputs. It does not interpret natural language, render UI, run external tools, or modify game/runtime state.

An approval is distinct from promotion. Imported content and assistant output cannot grant authority. Corrections append new evidence and state events; they never erase history. Any projection is rebuildable and non-authoritative.
